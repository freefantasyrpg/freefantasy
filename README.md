# Free Fantasy

**Free Fantasy** is a lightweight, class‑based fantasy role‑playing system inspired by  
**AD&D 2e**, **Basic Fantasy**, and modern OSR design.  
It incorporates additional mechanics from **The Black Hack**, **Traveller**, and original rules created specifically for this project.

The system aims to be:
- **Fast** – simple resolution mechanics and quick combat flow  
- **Compatible** – most published adventures and bestiaries work without modification  
- **Modular** – easy to expand, hack, or customize  
- **Classic** – familiar fantasy archetypes with modern refinements  

---
